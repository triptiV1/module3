# 🤖 Comprehensive NLP Chatbot Project

A sophisticated chatbot implementation using **four major NLP libraries**: NLTK, ChatterBot, TextBlob, and spaCy.

## 🚀 Quick Start

### Option 1: Easy Launch (Recommended)
```bash
python run_chatbot.py
```

### Option 2: Direct Launch
```bash
python comprehensive_nlp_chatbot.py
```

## 🔧 Manual Setup (if needed)

```bash
# Install dependencies
pip install -r requirements.txt

# Download spaCy model
python -m spacy download en_core_web_sm

# Run the chatbot
python comprehensive_nlp_chatbot.py
```

## 💡 Features

- **Multi-Library NLP**: Combines NLTK, ChatterBot, TextBlob, and spaCy
- **Sentiment Analysis**: Real-time emotion detection
- **Entity Recognition**: Identifies people, places, organizations
- **Intent Classification**: Understands conversation context
- **Learning Capabilities**: Adapts to your communication style
- **Analysis Reports**: Detailed NLP analysis on demand

## 🎯 Commands

- **Chat normally** - Natural conversation
- **`analyze [text]`** - Get NLP analysis
- **`stats`** - View conversation statistics  
- **`help`** - Show all features
- **`quit`** - Exit with summary
